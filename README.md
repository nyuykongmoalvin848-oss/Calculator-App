# Calculator Project

## Describtion

A calculator App built with HTML, CSS and JS

## Project Overview

This project replicates the look and basic behavior of the iOS calculator while focusing on...

Clear folder structure

Accurate calculation logic
Smooth and responsive user interactions
Handling common calculator edge cases

## Design Updates

Improved button press animations (scale and brightness effects)
More accurate spacing, rounding, and typography
Mobile-first responsive layout
Full-screen layout on small devices

## Functional Updates

Cleaner calculation logic
Improved operator handling
Better reset behavior after calculations
Safer division handling (division by zero)

## Features

Addition
Subtraction
Multiplication
Division
Chained operations
All Clear (AC)
Positive / Negative toggle
Percentage calculation
Decimal input handling
Keyboard input support
Mobile responsive layout

## Cases Handled

Prevents multiple decimal points in a number
Prevents invalid operator chaining
Division by zero returns Error
Proper reset after pressing equals
Backspace deletes digits correctly

HTML
CSS
